Creating a Map

To create a world it is recommended that you setup a bukkit server with worldedit & voxel sniper . It is also recommended that worlds are created in a empty flatworld. There are several ways to do this:

Option 1:
Download this level data file: level.dat
Create a empty world folder and place it in there. Then load the world. This level.dat uses the flatworld generator with this preset string: 2;0;1;4 You can also use this string directly in minecraft for the flatworld generator settings.

Option 2:
Install one of the bukkit void world generator plugins such as:
Bukkit - VoidGenerator

Note: Starting to build in a void world can be a-bit tricky if you don't have world edit or voxelsniper installed since theres nothing to build on. This can be remedied by using MCEdit to create a starter platform.

Flat World Generator Options

The flat world generator options string consists of four parts: version;blocks;biome;structures The only thing you should change from the void world settings is the biome.
